🌐 ¿Qué es Red Alset?

Red Alset es una plataforma descentralizada donde cada componente es un bloque Lego digital: autónomo, modular y accesible.

Principios:

  • Descentralización: Sin servidores centrales
  • Autonomía: Cada agente es dueño de sus datos
  • Composición: Los agentes se conectan como Legos
  • Inclusión: Cualquier persona puede participar

👤 Agentes

Un Agente es la unidad fundamental. Contiene: ID único, RootCID (contenido), Balance UTXO (economía), Estado neuronal (IA opcional).

(crear-agente "mi-tienda")
(register-name "mitienda.ans" "mi-tienda")
(set-agent-root "mi-tienda" "QmHash...")

🧩 Módulos como Piezas Lego

Cada módulo es un agente con un rol específico. Módulos predefinidos: Inventario, Ventas, Nómina, Finanzas, Clientes.

(connect-to-neuron "modulo-inventario" 0.8)

📊 Entidades Programáticas

Las entidades son objetos con atributos que pueden heredar y relacionarse.

(crear-entidad "Producto" '{"nombre":"Laptop","precio":1200})

Tipos de relación:

  • 1:1 - Uno a uno
  • 1:N - Uno a muchos
  • N:M - Muchos a muchos
  • Dependencia - Una entidad depende de otra
  • Herencia - Una entidad hereda de otra

🔗 Relaciones entre Entidades

Las relaciones definen cómo las entidades se conectan entre sí.

(crear-relacion "compra" "Cliente" "Producto" "1..*")

🔐 Seguridad estilo JWT en Alset

Tokens descentralizados basados en identidades criptográficas.

POST /api/auth/token
{
  "agent_id": "mi-agente",
  "roles": ["admin"],
  "duracion_horas": 24
}

Tienda Comunitaria:

(crear-agente "tienda")
(crear-entidad "Producto" '{"nombre":"Laptop","precio":1200})
(register-name "mitienda.negocio.ans" "tienda")
(generar-pagina-web "tienda")
